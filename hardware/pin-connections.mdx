SAVANT의 시스톨릭 어레이는 총 9개의 신호선으로 4개의 PE 보드를 연결합니다. 각 신호선은 CLOCK 핀과 DATA 핀으로 이루어진 쌍이거나 단독 타이밍 핀이며, 모든 연결에는 암-암(female-to-female) 점퍼 와이어를 사용하여 보드 핀을 직접 연결합니다. 연결 전에 보드 위치(PE 좌표)와 핀 번호를 충분히 숙지한 후 작업을 시작하세요.

## 보드별 핀 역할 표

아래 표는 각 보드에서 사용하는 핀과 그 역할을 정리한 것입니다. `→`는 해당 보드에서 신호를 **송신**하는 핀, `←`는 **수신**하는 핀을 의미합니다.

| 보드 | D2 역할 | D3 역할 | D4 역할 | D5 역할 | D6 역할 | A4 / A5 |
|------|---------|---------|---------|---------|---------|---------|
| PE(0,0) | CLOCK_A → PE(0,1).D2 | CLOCK_B → PE(1,0).D2 | DATA_A → PE(0,1).D4 | DATA_B → PE(1,0).D4 | TIMING → PE(1,1).D6 | — |
| PE(0,1) | CLOCK_A ← PE(0,0).D2 | CLOCK_B → PE(1,1).D3 | DATA_A ← PE(0,0).D4 | DATA_B → PE(1,1).D5 | — | — |
| PE(1,0) | CLOCK_B ← PE(0,0).D3 | CLOCK_A → PE(1,1).D2 | DATA_B ← PE(0,0).D5 | DATA_A → PE(1,1).D4 | — | — |
| PE(1,1) | CLOCK_A ← PE(1,0).D3 | CLOCK_B ← PE(0,1).D3 | DATA_A ← PE(1,0).D5 | DATA_B ← PE(0,1).D5 | TIMING ← PE(0,0).D6 | SDA(A4) / SCL(A5) |
| Single | — | — | — | — | — | — |

<Note>
PE(1,1)은 A 방향(PE(1,0)으로부터)과 B 방향(PE(0,1)으로부터) 두 가지 데이터를 동시에 수신해야 합니다. 이를 위해 D2와 D3 두 CLOCK 핀을 동시에 폴링하는 이중 수신 루프가 펌웨어에 구현되어 있습니다.
</Note>

## 전체 신호선 연결 체크리스트

연결이 완료될 때마다 아래 항목에 체크하세요. 한 번에 모든 선을 연결하기보다 쌍(CLOCK + DATA)별로 연결하고 동작을 확인하는 방식을 권장합니다.

### A값 가로 흐름 (PE(0,0) → PE(0,1))

- [ ] **연결 1**: PE(0,0).D2 → PE(0,1).D2 &nbsp;*(A 흐름 CLOCK)*
- [ ] **연결 2**: PE(0,0).D4 → PE(0,1).D4 &nbsp;*(A 흐름 DATA)*

### B값 세로 흐름 (PE(0,0) → PE(1,0))

- [ ] **연결 3**: PE(0,0).D3 → PE(1,0).D2 &nbsp;*(B 흐름 CLOCK)*
- [ ] **연결 4**: PE(0,0).D5 → PE(1,0).D4 &nbsp;*(B 흐름 DATA)*

### B값 세로 흐름 (PE(0,1) → PE(1,1))

- [ ] **연결 5**: PE(0,1).D3 → PE(1,1).D3 &nbsp;*(B 흐름 CLOCK)*
- [ ] **연결 6**: PE(0,1).D5 → PE(1,1).D5 &nbsp;*(B 흐름 DATA)*

### A값 가로 흐름 (PE(1,0) → PE(1,1))

- [ ] **연결 7**: PE(1,0).D3 → PE(1,1).D2 &nbsp;*(A 흐름 CLOCK)*
- [ ] **연결 8**: PE(1,0).D5 → PE(1,1).D4 &nbsp;*(A 흐름 DATA)*

### D6 타이밍 신호

- [ ] **연결 9**: PE(0,0).D6 → PE(1,1).D6 &nbsp;*(경과 시간 측정 타이밍)*

### GND 공통 연결

- [ ] PE(0,0) GND → 브레드보드 공통 GND 레일
- [ ] PE(0,1) GND → 브레드보드 공통 GND 레일
- [ ] PE(1,0) GND → 브레드보드 공통 GND 레일
- [ ] PE(1,1) GND → 브레드보드 공통 GND 레일
- [ ] Single GND → 브레드보드 공통 GND 레일 *(대조군도 공통 GND 연결)*

### LCD I2C 연결 (PE(1,1))

- [ ] PE(1,1).A4 → LCD SDA
- [ ] PE(1,1).A5 → LCD SCL
- [ ] PE(1,1) 5V → LCD VCC
- [ ] PE(1,1) GND → LCD GND

## 단계별 연결 절차

<Steps>
  <Step title="보드 배치 및 라벨 부착">
    4개의 PE 보드를 2×2 격자 형태로 브레드보드 또는 작업 판 위에 배치합니다. 각 보드 옆에 PE(0,0), PE(0,1), PE(1,0), PE(1,1) 라벨을 붙여 혼동을 방지하세요. 단일 보드(Single)는 격자와 분리된 위치에 놓습니다.
  </Step>
  <Step title="공통 GND 연결">
    배선 작업에 앞서 모든 보드의 GND 핀을 브레드보드 공통 GND 레일에 연결합니다. GND 공유가 없으면 이후 신호 연결을 테스트할 때 수신 오류가 발생할 수 있습니다.
  </Step>
  <Step title="A 흐름 1단계 연결 (PE(0,0) → PE(0,1))">
    점퍼 와이어 2개를 사용하여 연결 1(D2)과 연결 2(D4)를 완성합니다. 연결 후 PE(0,0)에서 간단한 `shiftOut` 테스트 스케치를 업로드하고, PE(0,1)에서 수신 값을 시리얼 모니터로 확인합니다.
  </Step>
  <Step title="B 흐름 1단계 연결 (PE(0,0) → PE(1,0))">
    연결 3(D3→D2)과 연결 4(D5→D4)를 완성합니다. 위와 같은 방법으로 PE(1,0)의 시리얼 모니터에서 수신 값을 검증합니다.
  </Step>
  <Step title="B 흐름 2단계 연결 (PE(0,1) → PE(1,1))">
    연결 5(D3→D3)와 연결 6(D5→D5)를 완성합니다. PE(0,1)의 DATA 핀(D5)과 PE(1,1)의 수신 핀(D5)이 일치하는지 핀 번호를 두 번 확인하세요.
  </Step>
  <Step title="A 흐름 2단계 연결 (PE(1,0) → PE(1,1))">
    연결 7(D3→D2)과 연결 8(D5→D4)를 완성합니다. PE(1,1)은 이 시점부터 두 방향(D2, D3)에서 CLOCK을 수신하게 됩니다. 이중 수신 폴링 루프가 정상 동작하는지 테스트 스케치로 확인합니다.
  </Step>
  <Step title="D6 타이밍 신호 연결">
    연결 9: PE(0,0).D6와 PE(1,1).D6를 점퍼 와이어로 연결합니다. 이 선은 연산 전체 경과 시간 측정에 사용되므로, 연결이 느슨하지 않도록 핀에 단단히 꽂습니다.
  </Step>
  <Step title="LCD 연결">
    PE(1,1)의 A4(SDA), A5(SCL), 5V, GND를 LCD 모듈의 해당 핀에 연결합니다. I2C 주소(기본값 `0x27` 또는 `0x3F`)를 확인하여 펌웨어 설정과 일치시킵니다.
  </Step>
  <Step title="전체 통전 전 최종 점검">
    모든 연결이 완료된 후 전원을 인가하기 전에 아래 점검 항목을 확인합니다.
    - 체크리스트의 9개 신호선 전부 체크되었는가
    - 5개 보드 GND 모두 공통 레일에 연결되었는가
    - 어떤 핀도 5V 전원 핀에 직접 연결되지 않았는가
    - 점퍼 와이어가 인접 핀에 단락되지 않았는가
  </Step>
</Steps>

## 흔한 실수 및 주의사항

### 보드 혼동

격자를 구성하는 4개의 보드는 외형이 동일합니다. 라벨 없이 작업하면 PE(0,1)에 꽂아야 할 선을 PE(1,0)에 꽂는 오류가 발생하기 쉽습니다. 반드시 각 보드에 라벨을 부착하고, 배치를 바꾸지 않도록 테이프 등으로 고정하는 것을 권장합니다.

### 핀 방향 오류

<Warning>
송신 핀(OUT)을 다른 보드의 송신 핀(OUT)에 연결하면 두 출력이 충돌하여 보드가 손상될 수 있습니다. 반드시 한쪽은 송신, 반대쪽은 수신 핀인지 위의 핀 역할 표를 참조하여 확인하세요.
</Warning>

### CLOCK과 DATA 혼동

CLOCK 선과 DATA 선을 바꿔 꽂으면 데이터 수신 오류가 발생하지만, 하드웨어 손상은 없습니다. 오동작 시 가장 먼저 CLOCK/DATA 선 교체를 의심하세요. 점퍼 와이어의 색상을 CLOCK과 DATA에 일관되게 구분하면(예: 노란색 = CLOCK, 파란색 = DATA) 오류를 줄일 수 있습니다.

### PE(1,1)의 혼용 핀 주의

PE(1,1)은 A 방향 CLOCK을 D2, B 방향 CLOCK을 D3에서 수신합니다. 반면 PE(0,1)과 PE(1,0)은 각각 D3를 CLOCK 송신에 사용합니다. 핀 번호가 비대칭이므로 표를 참조하지 않고 기억에 의존하는 연결은 오류 위험이 높습니다.

<Tip>
연결 완료 후, 각 보드에 핀 번호와 방향을 출력하는 간단한 진단 스케치를 업로드하여 실제 신호 흐름을 확인하면 전체 시스템 테스트 전에 오배선을 발견할 수 있습니다.
</Tip>
